## Команда

- Сейит  каталог, фильтры, сортировка
- Денчик  карточка товара, корзина
---

## Стек
HTML, CSS, JavaScript (ES6+). Без фреймворков. Состояние корзины хранится в localStorage.

## Что реализовано
**Каталог**
- Карточки товаров с названием, ценой, рейтингом и категорией
- Сортировка по названию и цене
- Фильтр по диапазону цены и по рейтингу
- Добавление в корзину без перехода на другую страницу

**Карточка товара**
- Слайдер с миниатюрами
- Счётчик количества перед добавлением
- Аккордеон с техническими характеристиками
- Блок похожих товаров

**Корзина**
- Список товаров с управлением количеством
- Пересчёт суммы в реальном времени
- Удаление отдельных позиций
- Промокод SAVE10 даёт скидку 10%
- Пустое состояние с кнопкой возврата в каталог

**Общее**
- Адаптивная вёрстка от 375px до 1440px
- Badge на иконке корзины обновляется при любом изменении
- Корзина сохраняется при перезагрузке страницы
